difference
Finder forskellen mellem to objekter.
Kan kun benyttes på relativt simple objekter
Er lavet til logbogen (render)

Details
difference(oldObjects, newObjects, keyName)
Example code
var old = [{ id: 1 }, { id: 7 }, { id: 8 }];
var ny = [{ id: 2 }, { id: 7 }, { id: 9 }, { id: 19 }, { id: 29 }, { id: 39 }];
var dif = Objects.difference(old, ny, "id");
Parameters
oldObjects : Array
Array med objekter

newObjects : Array
Array med objekter

keyName : String
Den key der skal sammenlignes på

Return value
sObject
