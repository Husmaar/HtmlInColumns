Public properties
RENDERED Name of the event that fires when all the columns is rendered
nonLastTag Tags that cant be in the end of a column
nonSplitTag Tags that cant be divided over two columns
options.columnClass CSS class for the column divs. Default is column
options.containerID The dom-id of the container div. Default is #Container
options.protectAgainstOrphans Should the script protect against orphans
Hvis denne er sat til true, vil tekst kunne blive højere end maxheight, dvs højere end containerens højde.
Default er true
orphansLength The number of char that defines a orphan
uuid UUID der bruges til at lave spalte id. Sættes før alle spalternes dom id, f.eks. 42b5e564_1df2_12ac_130fe29d9645_.
